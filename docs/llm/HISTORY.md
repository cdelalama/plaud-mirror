Added two tests: `parseByteRange` unit test covering the four RFC 7233 single-range shapes (`bytes=0-999`, `bytes=N-`, `bytes=-N`, clamped over-end) plus garbage rejection, and an HTTP-level test that exercises the full 200, 206, suffix-form, and 416 paths.
